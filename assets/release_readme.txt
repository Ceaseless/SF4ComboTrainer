sf4combotrainer readme

- this program may not run if you don't have a .NET runtime environment installed
this is probably a non issue since you most likely already have it but if it is check out http://msdn.microsoft.com/library/5a4x27ek(v=vs.110).aspx

- uncheck the steamversion check box if you are using the retail (non-steam) version of street fighter

- set your streetfighter keyboard 1 settings as follows or edit sf4keyboard.cfg
arrow keys for directions
NUM4 - 6 lp mp hp
NUM1 - 3 lk mk hk

- don't! have your graphic setting for stage quality set to low (black training room)
